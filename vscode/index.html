Lista de extensões para VSCode que podem ajudar a economizar tempo e aumentar a eficiência da codificação.

Conceito

O Visual Studio Code é um dos editores de código-fonte mais populares e amplamente utilizados entre os desenvolvedores.

Parte 1: Dividir a tela

Para usar o recurso de dividir tela no VSCode, basta apertar Ctrl + 2. Dessa forma, uma segunda tela aparece do lado da primeira. E podemos mover arquivos pra serem visualizados nessa segunda dela.

Parte 2: Mostrar em tela apenas o importante

Para se focar no código, pressione Ctrl + J pra ocultar o terminal e Ctrl + B pra ocultar a barra de navegação. Quando precisar dos recursos, pressione novamente as mesmas teclas.

Parte 3: Navegação

Para procurar e abrir arquivos, pressione Ctrl + P, e digite o nome ou parte do nome do arquivo para pesquisar dentro da pasta do projeto.

Parte 4: Navegando entre terminais

Mas e se eu tiver 5 terminais abertos no meu VSCode, como posso navegar entre terminais, use Ctrl + PgUp ou Ctrl + PgDown.
